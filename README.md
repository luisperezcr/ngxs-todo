# Angular Todo with Ngxs

This project is a simple todo app built with Angular, using Ngxs for state management.

## Technologies used
* Angular
* Ngxs (state management)
* Flex Layout
* Angular Material

You can see the app [here](https://gd-simple-todos.firebaseapp.com/).
